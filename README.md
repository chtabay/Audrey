# Maison Castel

Projet immobilier locatif commercial — 1 place de la Mairie, 31180 Castelmaurou.

Consultez le site du projet : **[Maison Castel — GitHub Pages](https://guillaumedeslandes.github.io/Audrey/)**

## Structure

- `01 - Inputs/` — Documents sources (avis de valeur, chiffrages, plans financiers)
- `02 - Plan/` — Livrables d'analyse (marché, juridique, urbanisme, business plan)
- `docs/` — Source du site web (MkDocs Material)

## Développement local

```bash
pip install -r requirements.txt
mkdocs serve
```
